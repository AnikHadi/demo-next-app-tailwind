# First, clone the Repository:

```bash
git remote add origin https://github.com/HadiAnik/demo-next-app-tailwind.git
```

# Second, Install all require dependencies or Package:

```bash
npm install
```

# Third, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
